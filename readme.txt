Email bbPress topic/reply mentioned users, set custom email and some basic settings

== Description ==

Does the same functionality as BuddyPress, emailing users who get mentioned in a bbPress topic or reply. The aim of this plugin is when to enable that feature when no BuddyPress is there.

== Installation ==

* Install and activate the plugin:

1. Upload the plugin to your plugins directory, or use the WordPress installer.
2. Activate the plugin through the \'Plugins\' menu in WordPress.
